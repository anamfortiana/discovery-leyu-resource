# discovery-leyu-resource

这是一个用于归档和发布多个独立 HTML 页面的资源仓库。仓库中的页面彼此独立，不针对任何特定域名或网站，主要用于记录、展示或提供各类静态 HTML 内容。

## 项目简介

本项目旨在整理和保存一系列独立的 HTML 页面，方便统一管理和访问。每个页面均为自包含的静态文件，不依赖外部服务或特定运行环境，可直接在浏览器中打开查看。

## 目录结构

```
discovery-leyu-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面可能引用的公共资源（如样式、图片、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，命名清晰，便于识别。
- `assets/`：存放页面共用的静态资源，按需引用。

## 页面归档说明

- 所有页面均为静态 HTML，不包含动态后端逻辑。
- 页面内容可能涉及多种主题，彼此之间无直接关联。
- 归档的目的是便于长期保存和查阅，不保证所有页面持续更新。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改独立页面。
- 欢迎通过 Issue 或 Pull Request 提出建议或提交新的页面。
- 提交页面时请确保内容合规，并尽量保持文件结构清晰。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 下的任意 HTML 文件即可查看。
3. 也可将整个仓库部署到任何静态托管服务上，通过 URL 访问对应页面。

## 注意事项

- 本仓库仅用于归档和展示，不保证页面中链接或资源的长期可用性。
- 页面内容不代表维护者立场，仅供技术或存档用途。
